# PROPOSAL.md

## What I'm building
I am building an interactive map-based web app that shows real-world data (like weather or locations) and lets users explore it visually.

## Which API I'm using
Open-Meteo API  
https://open-meteo.com

## Why I chose this
I chose this API because it provides real-time data and is easy to use without an API key. I also think maps and live data are interesting and useful for users.

## Core features
- Display a map with data points (like weather or locations)
- Show a heatmap or color-based visualization
- Allow users to search for a location
- Click on a location to see more details
- Show loading and error messages when fetching data

## What I don't know yet
- How to properly use `fetch()` with async/await
- How to work with JSON data from an API
- How to connect API data to a map or heatmap visualization
